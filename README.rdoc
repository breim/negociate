== negociate

Sistema de cadastro de empresas para realização de emprestimos e recebimentos da empresa.

É adaptado para o mundo mobile (bootstrap 3), tendo em vista que o credor do emprestimo fica na rua lançando o emprestimo/recebimento do valor.
Tem como objetivo estabeleciamentos comerciais, pois o dono do emprestimo, não tem o controle de quem pagou diariamente o valor devido.

O sistema ainda não está completo, devido a falta de posicionamento do cliente.
